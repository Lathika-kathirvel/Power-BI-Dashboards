Sales Dashboard Analysis (Power BI)


Description

This project consists of 8 interactive dashboards created using Microsoft Power BI to analyze sales data across different platforms and regions. The dashboards provide insights into sales performance, profit trends, customer behavior, and product categories.


File Information

File Name: lathika.pbix

Tool Used: Microsoft Power BI Desktop

Total Dashboards: 8


Dashboards Included

1.Amazon Sales Dashboard

2.Adidas Sales Dashboard

3.ConnectX Dashboard

4.InnovateX Dashboard

5.Pizza Sales Dashboard

6.Movies Dashboard

7.Retail India Dashboard

8.UK Road Dashboard


Key Features

Interactive KPI cards (Sales, Profit, Quantity, Discount)

Slicers for filtering data dynamically

Category and sub-category analysis

Time-based analysis (Year and Month trends)

Region and city-level insights

Customer segmentation

Branch-wise and payment-based analysis


Key Metrics

-Total Sales

-Total Profit

-Total Quantity

-Discount Percentage

-Tax Calculation (where applicable)

-Ratings and Customer Type

Insights Generated
